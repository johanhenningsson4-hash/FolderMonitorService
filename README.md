# FolderMonitorService

## Purpose
The FolderMonitorService is a .NET Framework 4.7.2 application designed to monitor changes in a specified folder. It tracks file creations, modifications, and deletions, providing a reliable way to observe folder activity.

## Features
- Real-time folder monitoring
- Logs file changes (create, modify, delete)
- Configurable folder paths
- Lightweight and efficient

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/johanhenningsson4-hash/FolderMonitorService.git
   ```
2. Open the solution in Visual Studio.
3. Build the project to restore dependencies and compile the application.
4. Configure the folder path to monitor in the application settings.
5. Run the application to start monitoring folder changes.
